# Node.js-MSSQL

## Description

A standard software project.

## Installation

1.  Clone the repository: `git clone https://github.com/your-username/nodejs-mssql.git`
2.  Navigate to the project directory: `cd nodejs-mssql`
3.  Install dependencies: `npm install`
4.  Configure MSSQL connection:  Edit `src/config.js` and set the connection details (server, database).
5.  Run: `npm start`

## Usage

1.  **Configuration:**  Configure the connection details in `src/config.js` (server address, database name, user, password).
2.  **Run the Application:** Execute `node src/main.js` to start the application.
3.  **Testing:** Use a SQL client (e.g., DBeaver, SQL Developer) to connect and verify data retrieval and manipulation.
